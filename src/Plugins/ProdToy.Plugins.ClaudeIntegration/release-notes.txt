Claude Shortcuts — generic launch profiles
  - New "Launch profile" dropdown at the top of the edit form (defaults to Claude CLI). Profiles: Claude CLI, npm, Vite (via npx), dotnet, PowerShell, Custom command.
  - Switching the profile swaps the args label, hint text and default args; the args box is only overwritten if empty or still holding the previous profile's default, so your edits survive.
  - Launcher now runs the profile's command (claude / npm / npx / dotnet / pwsh), not a hardcoded `claude`. Custom profile treats the args box as the full command line. The "(X --continue || X)" retry wrapper stays Claude-only.
  - The shortcut row's footer now shows the profile's command (e.g. "wt · Command Prompt  •  npm run dev" or "wt · Command Prompt · tab  •  claude --continue").
